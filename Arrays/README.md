# Arrays in Java

This folder contains array-based programs implemented in Java as part of my Java Fundamentals learning journey.

The goal of these programs is to build a strong understanding of array operations, searching, sorting, traversal, and basic problem-solving before moving to advanced Data Structures and Algorithms.


## Topics Covered

### Basic Operations
- Print Array
- Copy Array
- Merge Two Arrays
- Reverse Array

### Searching
- Linear Search

### Sorting
- Manual Array Sorting

### Array Calculations
- Sum of Elements
- Average of Array

### Element Operations
- Find Maximum Element
- Find Minimum Element
- Second Largest Element
- Second Smallest Element

### Frequency & Duplicates
- Count Frequency of Elements
- Count Even and Odd Elements
- Duplicate Elements

### Array Manipulation
- Insert Element
- Delete Element
- Left Rotate Array by One Position
- Right Rotate Array by One Position

### Array Validation
- Check Array Equality
- Check Sorted Array

## Concepts Practiced

- Arrays
- Loops
- Conditional Statements
- Nested Loops
- User Input
- Array Traversal
- Swapping
- Searching
- Sorting Logic
- Time Complexity Basics

## Learning Outcome

After completing these programs, I gained confidence in:

- Performing common array operations

- Writing optimized loop logic
