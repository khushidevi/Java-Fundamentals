# Java Strings

## About

This folder contains Java programs that demonstrate fundamental string manipulation and problem-solving techniques. These programs focus on understanding string operations, character processing, searching, comparison, and logical problem solving without relying heavily on built-in library methods.

## Programs Included

- Anagram Strings
- Check Palindrome String
- Check Rotation of String
- Check Substring
- Compare Strings
- Count Occurrence of Character
- Count Vowels and Consonants
- Count Words in Sentence
- Count Digits, Alphabets and Special Characters
- Email to Username
- First Non-Repeating Character
- Length of String
- Longest Word in Sentence
- Remove Duplicate Characters
- Remove Whitespace
- Replace Character
- Replace Word
- Reverse String
- Reverse Words
- Toggle Case

## Concepts Practiced

- String Traversal
- Character Manipulation
- String Comparison
- String Searching
- Character Frequency Counting
- Nested Loops
- StringBuilder
- Conditional Statements
- User Input using Scanner
- Logical Problem Solving

## Language

- Java

## Learning Outcome

These programs helped me build a strong understanding of Java Strings, improve problem-solving skills, and learn how common string operations work by implementing the logic manually before using advanced library methods.
